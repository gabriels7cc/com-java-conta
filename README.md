# Projeto Cofre em Java

Projeto desenvolvido em Java com foco em praticar fundamentos essenciais da linguagem e lógica de programação orientada a objetos.

Este projeto simula um cofre digital acessado via console, permitindo operações básicas de controle de saldo e histórico de transações.

---

## Funcionalidades

O sistema permite ao usuário:

- Guardar dinheiro  
- Retirar dinheiro  
- Consultar saldo atual  
- Visualizar extrato das operações  
- Validação de valores inválidos  

---

## Conceitos praticados

Este projeto aplica:

- Criação de classes e métodos
- Encapsulamento de dados
- Estruturas condicionais (`if / else`)
- Estruturas de repetição (`do-while`)
- Manipulação de variáveis
- Interação com usuário via `Scanner`
- Validações de regras de negócio

---

## Estrutura do Projeto

O projeto é composto principalmente por:

- **Cofre.java** → Responsável pelas regras de negócio e operações financeiras
- **Programa.java** → Responsável pela interface via console e interação com o usuário
